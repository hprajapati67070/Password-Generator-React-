# Basic Password Generator

A simple, interactive, and responsive Password Generator web application built with **React**, **Vite**, and **Tailwind CSS**.

## Features

- **Dynamic Generation**: Automatically generates a new password whenever the length, number option, or character option changes.
- **Customizable Length**: Adjust password length from 6 to 16 characters using a slider.
- **Custom Options**: Toggle the inclusion of numeric digits (`0-9`) and special characters (`!@#$%^&*-+`).
- **Copy to Clipboard**: Copy the generated password instantly to your clipboard with a single click.

## Technologies Used

- [React](https://react.dev/) (Hooks: `useState`, `useEffect`, `useCallback`)
- [Vite](https://vite.dev/) (Fast frontend tooling)
- [Tailwind CSS](https://tailwindcss.com/) (Utility-first CSS styling)

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository or download the project files.
2. Navigate to the project root directory:
   ```bash
   cd yourPath
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the App

To run the application locally in development mode:

```bash
npm run dev
```

Open your browser and navigate to the local URL provided by Vite (usually `http://localhost:5173`).
